## üÜï NEW IN v2.2: PATH CONVENTIONS

### **Project-Relative Path Standards**

#### **For Project Files:**

```python
# Project: Project_Startup
Path: Project_Startup/src/main.py               # ‚úÖ Correct
Path: /home/herb/Desktop/Project_Startup/src/main.py  # ‚ùå Wrong

# Project: CSM  
Path: CSM/enhanced_claude_monitor.py            # ‚úÖ Correct
Path: /home/herb/Desktop/CSM/enhanced_claude_monitor.py  # ‚ùå Wrong
```

#### **For Shared Infrastructure:**

```python
# Project_BaseFiles (shared across projects)
Path: Project_BaseFiles/Scripts/ScriptMenu.py   # ‚úÖ Correct
Path: ~/Desktop/Project_BaseFiles/Scripts/ScriptMenu.py  # ‚ùå Wrong
```

#### **For Generated Templates:**

```python
# Templates should use placeholder for project name
Path: {{project_name}}/README.md               # ‚úÖ Template
Path: Project_BaseFiles/templates/README_template.md  # ‚úÖ Template source
```

### **Path Resolution Rules**

1. **Project Base**: The directory containing the project (e.g., `Project_Startup/`)
2. **Relative Reference**: All paths relative to their project base
3. **Cross-Project Links**: Use full project name as base (e.g., `Project_BaseFiles/`)
4. **Template Variables**: Use `{{project_name}}/` for generated content
